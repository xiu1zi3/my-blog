## 什么是jQuery的Ajax？

**jQuery的Ajax**是jQuery库提供的一种简化异步网络请求的方法，它封装了原生JavaScript的XMLHttpRequest对象，使开发者能够更方便地与服务器进行数据交互。在Web开发中，Ajax（Asynchronous JavaScript and XML）技术允许在不刷新整个页面的情况下，与服务器交换数据并更新部分页面内容，从而提升用户体验。

## 后端实现

以下是后端SpringBoot控制器的实现，用于提供 bot 信息的API接口：

```java
RequestMapping("/pk/")
public class BotInfoController {
  @RequestMapping(" getbotinfo/")
  public Map<String , String> getBotInfo(){
    Map<String,String> bot1 = new HashMap<>();
    bot1.put( "name " , "bot1");
    bot1.put( "rating "，"1500");
    return bot1;
  }
}
```

**说明**：
- `@RestController`注解表示这是一个REST风格的控制器，会自动将返回值转换为JSON格式
- `@RequestMapping("/pk/")`定义了控制器的基础路径
- `getBotInfo()`方法返回一个包含机器人名称和评分的Map，Spring会自动将其转换为JSON响应

## 前端实现

在Vue 3组件的`setup()`函数中，我们使用jQuery的`$.ajax()`方法向后端发送请求：

```javascript
setup:() => {
  let bot_name = ref("");
  let bot_rating = ref("");
  
  // 使用jQuery的Ajax方法发送GET请求
  $.ajax({
    url:"http://127.0.0.1:3000/pk/getbotinfo/",
    type:"get",
    success:resp=>{
      console.log(resp);
      bot_name.value = resp.name;
      bot_rating.value = resp.rating;
      }
    });
    return{
      bot_name,
      bot_rating
};
```
`setup:()`可以认为是整个组件的入口





用户浏览器拿到了结果：

```javascript
success:resp=>{
console.log(resp);
bot_name.value = resp.name;
bot_rating.value = resp.rating;
}
});
```

通过这种方式，数据被动态绑定到前端页面中，实现了前后端的无缝交互。
