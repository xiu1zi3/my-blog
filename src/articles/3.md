# VS Code 插件推荐

## 推荐一些提高开发效率的 VS Code 插件

### 代码提示
好的代码提示插件可以提高编码效率和准确性。

- **IntelliSense for CSS class names**: 智能提示 CSS 类名
- **ESLint**: 代码质量检查
- **Prettier**: 代码格式化

### 版本控制
版本控制是开发过程中的重要环节，相关插件可以提高工作效率。

- **GitLens**: 增强 Git 功能
- **GitHub Pull Requests and Issues**: 直接在 VS Code 中管理 PR 和 Issues

### 其他实用插件

- **Live Server**: 本地开发服务器
- **REST Client**: 直接在 VS Code 中发送 HTTP 请求
- **Docker**: 容器管理
