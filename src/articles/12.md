### docker相关
检查正在运行的docker
```shell
docker ps
```

检查创建的docker，既包括正在运行的docker，也包括其他状态的docker
```shell
docker ps -a
```

### Java项目相关
启动主服务

```shell
nohup /usr/local/java/jdk17.0.8.1/bin/java \
-Dshort-link.domain.default=这里填服务器公网IP:8003 \
-jar /home/acs/shortlink/shortlink-aggregation.jar > logs/shortlink-aggregation.file 2>&1 &
```

比如公网ip 是 `106.14.149.83` ，那么命令就是

```shell
nohup /usr/local/java/jdk17.0.8.1/bin/java \
-Dshort-link.domain.default=106.14.149.83:8003 \
-jar /home/acs/shortlink/shortlink-aggregation.jar > logs/shortlink-aggregation.file 2>&1 &
```

启动网关服务

```shell
nohup /usr/local/java/jdk17.0.8.1/bin/java \
-jar /home/acs/shortlink/shortlink-gateway.jar > logs/shortlink-gateway.file 2>&1 &
```

### 其他

故障排查可能用得上以下命令：
查找与 `xxx` 有关的进程
```shell
ps -ef | grep xxx
```
比如：查找与`shortlink-gateway.jar`有关的进程
```shell
ps -ef | grep shortlink-gateway.jar
```
查看存储空间
```shell
free -h
```
